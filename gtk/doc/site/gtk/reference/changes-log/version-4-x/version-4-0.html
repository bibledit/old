Version 4.0

12 December 2009

After export to OpenDocument, if any markers were unformatted, the user gets notified of that.

The --debug argument can be passed on the command line to allow debugging in gdb.

The relevant files of the GoBibleCreator were included with the Bibledit package.

All the windows can be resized and can float freely within the Bibledit frame.

Chmsee is no longer needed because the USFM documentation is now available in .htm files.

A couple of bugs was fixed. Hopefully yours too!

When a note was created or edited, and this note would normally not have displayed, it will display on that single occasion. This avoids a situation that a user creates or edits a note, then saves it, then it disappears.

Next-generation Bible text editor replaces the current one. Expect perfect undo and redo operations, and other goodies.
