Restore

Restore Assistant

The Restore Assistant is available through menu File / Restore. You can restore a Bible, or the Notes, of Everything. When restoring everything, the data will be available after restarting Bibledit. More information is available with the Backup Assistant, as Restore and Backup complement each other.
