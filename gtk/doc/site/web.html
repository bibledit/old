3 Web

Bibledit-Web is a web application for translating the Bible. It is part of the Bibledit suite.
A demo is available at http://bibleconsultants.dyndns.org/bibledit-web-demo. You can login there. Use username "manager" and password "manager", or username "translator" and password "translator", or username "consultant" and password "consultant", or username "member" and password "member".You can also create your own account, then temporarily login as "manager" and upgrade your account to the level you desire, then use your own account from then on.
More information is available at http://bibleconsultants.dyndns.org/bibledit-web-demo/help/index.php.
Translations are being done on https://launchpad.net/bibledit, your help is welcomed.
